# Music-Recommendation-Engine
A Python music recommendation engine using audio feature similarity (Cosine, Euclidean, Pearson, Manhattan) with a Tkinter GUI — paired with an R-based EDA report.

It analyses a Spotify-style music dataset and recommends tracks or artists based on audio feature similarity. Users can interact with the system through a clean Tkinter GUI, choosing from four different similarity algorithms.

An accompanying EDA report (written in R) explores the dataset's structure, distributions, and correlations — with a particular focus on how musical features relate to song popularity.
